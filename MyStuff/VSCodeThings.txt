Step-by-Step: Add Custom Snippets in VS Code
1. Open User Snippets
- Press Ctrl+Shift+P (or F1) to open the Command Palette.
- Type: Preferences: Configure User Snippets (no good)
- Type:	Snippets: Configure Snippets
- Select:
- powershell.json for PowerShell-specific snippets, or
- New Global Snippets file for cross-language snippets

2. Define (see example in [powershell.json])
3. Save

Pro Tips
- Use $TM_FILENAME or $CURRENT_YEAR for dynamic values
- You can create language-specific snippets for PowerShell, JSON, Markdown, etc.
- Organize snippets by function type: logging, diagnostics, headers, etc.
